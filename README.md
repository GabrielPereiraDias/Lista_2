# 🚀 Portfólio de Lógica e Algoritmos

Este repositório contém uma coleção de desafios de programação desenvolvidos para consolidar conceitos de estrutura de dados, variáveis e fluxos de controle.

## 🛠️ Tecnologias e Ferramentas
Para este trabalho, utilizei um conjunto de ferramentas que permitem a visão teórica e prática da programação:

* **[Draw.io](https://app.diagrams.net/):** Para a criação dos **Fluxogramas**. Uma etapa essencial para planejar a lógica antes da codificação.
* **[Visualg](https://visualg3.com.br/):** Para a implementação dos algoritmos em **Portugol (.alg)**, permitindo a simulação real dos cálculos e desvios condicionais.

---

## 📂 Projetos Realizados

Cada pasta abaixo representa um módulo completo, contendo o diagrama lógico e o código fonte:

| Pasta | Descrição Técnica | Status |
| :--- | :--- | :---: |
| `Calculadora de terreno` | Processamento de medidas para cálculo de área e custos. | ✅ |
| `IMC` | Algoritmo de saúde para cálculo de massa corporal e diagnóstico. | ✅ |
| `Média Trapézio` | Aplicação de fórmulas matemáticas em geometria plana. | ✅ |
| `Médias` | Sistema de entrada de dados e média aritmética acadêmica. | ✅ |
| `Soma` | Lógica básica de operadores. | ✅ |
| `Temperatura` | Conversor de escalas termométricas (Cº para Fº). | ✅ |

---

## 📁 Estrutura Padrão
Em cada diretório, você encontrará:
1.  **Diagrama de Fluxo:** Arquivo visual gerado no Draw.io demonstrando o caminho dos dados.
2.  **Código-Fonte:** Arquivo `.alg` pronto para rodar no Visualg.
